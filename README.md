# Командний проєкт Rozetka

Новий мобільний застосунок створюється окремо від попередніх напрацювань. Поточний перший крок: базовий екран входу для теми `Rozetka`.

## Що вже є

- чистий Expo + TypeScript каркас;
- стартовий екран входу;
- базова валідація `email + password`;
- локальне збереження сесії після входу;
- окремий екран після успішного входу;
- вихід з акаунта;
- окремий документ із першим таском у [docs/first-task-login.md](/Users/radonnazar/Documents/New%20project/Командний%20проєкт%20Rozetka/docs/first-task-login.md).

## Запуск

```bash
npm install
npm start
```
